# Solving 99 Haskell Problems 

Use ```:l main.hs``` in ghci to load all functions that solves problems.

## Problem 01
(*) Find the last element of a list.

``` haskell
*Main> p01 [1, 2, 3]
3
```

## Problem 02
(*) Find the last but one element of a list.

``` haskell
*Main> p02 [1, 2, 3, 4]
3
```
